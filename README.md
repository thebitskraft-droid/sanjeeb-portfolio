# Sanjeeb Dhakal - Professional Portfolio

This is a premium, high-performance professional portfolio website designed for **Sanjeeb Dhakal**, a Civil Engineer specializing in Road Quality Assurance.

## 🚀 How to Run in Visual Studio Code

1. **Open the Folder**: Open Visual Studio Code and go to `File > Open Folder...`. Select the `sanjeeb-portfolio` folder.
2. **Install Live Server (Recommended)**:
   - Go to the **Extensions** view in VS Code (Ctrl+Shift+X).
   - Search for **"Live Server"** and install the one by Ritwick Dey.
3. **Launch the Site**:
   - Open `index.html`.
   - Right-click anywhere in the editor and select **"Open with Live Server"**.
   - Your browser will automatically open the website.

## 📁 File Structure

- `index.html`: Home page with hero section and expertise overview.
- `about.html`: Professional biography and skills.
- `services.html`: Detailed service offerings.
- `projects.html`: Portfolio of completed works.
- `contact.html`: Contact form and professional details.
- `style.css`: The "brain" of the design. All colors, spacing, and typography are defined here.
- `assets/images/`: Contains high-quality engineering imagery used across the site.

## ✍️ How to Update Content

- **Updating Text**: Open any `.html` file and look for the text you want to change (e.g., within `<h1>`, `<p>`, or `<h3>` tags). Edit the text and save the file.
- **Updating Images**: Replace the images in the `assets/images/` folder with your own photos. Ensure they have the same filenames (e.g., `hero.png`) or update the `src` attribute in the HTML files.
- **Changing Colors**: Open `style.css` and modify the variables at the top under `:root`. For example, change `--primary` to your preferred brand color.

## 🛠 Technical Features

- **Responsive Design**: Works perfectly on Mobile, Tablet, and Desktop.
- **Modern Animations**: Uses AOS (Animate on Scroll) for smooth reveal effects.
- **Lucide Icons**: High-quality, scalable vector icons.
- **SEO Ready**: Proper meta tags and semantic HTML structure.
- **Zero Dependencies**: Runs immediately without needing `npm install` or complex build tools.
